# Temp Scripts

Scripts y archivos temporales utilizados durante el desarrollo y carga de datos.

## Contenido

- **actualizar_*.py**: Scripts de actualización de datos específicos
- **cargar_*.py**: Scripts de carga de datos desde PDFs y otras fuentes
- **corregir_*.py**: Scripts de corrección de datos
- **verificar_*.py**: Scripts de verificación de datos
- **validar*.py**: Scripts de validación
- **diagnosticar_*.py**: Scripts de diagnóstico
- **analizar_*.py**: Scripts de análisis
- **comparar_*.py**: Scripts de comparación
- **explore*.py**: Scripts de exploración
- **exportar_*.py**: Scripts de exportación
- **extraer_*.py**: Scripts de extracción
- **fix_*.py**: Scripts de corrección rápida
- **recalcular_*.py**: Scripts de recálculo
- **reload_*.py**: Scripts de recarga
- **reparsear_*.py**: Scripts de re-parseo
- **check_*.py**: Scripts de verificación rápida
- **reporte_*.py**: Scripts de generación de reportes
- **completitud*.py**: Scripts de análisis de completitud
- **Archivos .md**: Documentación temporal de reportes y guías
- **Archivos .yaml**: Archivos de configuración temporales
- **Archivos .json**: Archivos de mapeo temporales

## Nota

Estos scripts NO son parte del pipeline principal y se mantienen aquí solo como referencia.
No utilizar sin comprender el contexto en el que fueron creados.

## Pipeline Principal

El pipeline principal está en la raíz del proyecto:
- `pipeline.py`: Pipeline principal
- `bank_pipeline.py`: Pipeline para bancos
- `afp_pipeline.py`: Pipeline para AFPs
- `ratios.py`: Cálculo de ratios
- `normalizer.py`: Normalización de datos
- `xbrl_parser.py`: Parser de XBRL
- `cmf_api.py`: Cliente API CMF
- `query.py`: Utilidades de consulta
